MuTeBench Human Evaluation

Thank you for participating! This evaluation studies the quality of MIDI compositions generated by AI systems. It takes approximately 20-30 minutes. Please use headphones if possible.

Instructions:
  1. Listen to each audio sample (rendered from MIDI)
  2. Read the task description that was given to the AI
  3. Rate each sample on three criteria (1-5 scale)
  4. For A/B comparisons, simply choose which one sounds better

Rating Criteria

Criterion1 (Poor)3 (Acceptable)5 (Excellent)
Musical Correctness
Are the key, tempo, time signature, and harmony correct?		 Wrong key/tempo, dissonant, broken Mostly correct, minor issues Accurate, well-formed harmony
Structural Coherence
Does it have a meaningful structure? Does it sound like a complete piece?		 Random notes, no structure Some structure, but unfinished feel Clear sections, satisfying form
Overall Quality
Would you enjoy listening to this? Is it musically interesting?		 Unpleasant, would skip Decent, acceptable background Enjoyable, would listen again
