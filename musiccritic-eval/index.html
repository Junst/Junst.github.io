How to participate

1. Click a study below.
2. Use headphones and listen to each audio pair.
3. Rate Quality (1-5), Match (1-5), and Preference (A/B/Tie).
4. Click "Submit" — a JSON file will download automatically.
5. Send the JSON file to the organizer. Thank you!
