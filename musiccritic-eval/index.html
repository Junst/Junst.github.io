MusicCritic Human Evaluation

Welcome! Please use headphones in a quiet environment. Each set uses blinded A/B comparisons with Quality MOS, Match MOS, and Preference ratings. After completing, click Download Results and send the JSON file to the researchers.

Evaluation Sets

1. Feature vs Audio Critic

23 pairs ~25 min

Direct A/B comparison between feature-selected vs audio-selected candidates across 10 genres.

2. MusicGen Backend

6 pairs ~10 min

Baseline vs critic-selected on MusicGen-Large backend.

3. Multi-Prompt Robustness

4 pairs ~5 min

Baseline vs selected across different text prompts for the same genre.
